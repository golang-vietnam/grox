# Golang VN Grok

This is an experiment project of Golang VN team.

## Prerequisite

- [RethinkDB](http://rethinkdb.com/docs/install/)

## API Server

### Install dependencies

```
make install
```

### Build

```
make build
```

### Start

```
$GOPATH/bin/grox-server
```

## Frontend

### Install dependencies

```
npm install
```

### Development

```
npm run dev
```

### Production

```
npm run build
npm start
```
